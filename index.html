Connect Google Sheets

This connects the app to a shared Google Sheet so your whole team reads and writes the same live data. Setup takes about 3 minutes.

SETUP STEPS
  1. Go to script.google.com → New Project
  2. Delete any existing code, paste the Apps Script below
  3. Click Deploy → New Deployment → Web App
  4. Set Execute as: Me · Who has access: Anyone
  5. Click Deploy → copy the Web App URL
  6. Paste that URL below and click Save
